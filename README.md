# GPU SKU Upgrade TCO Calculator

Static browser preview for the GPU SKU Upgrade TCO Calculator.

## Open

Open `index.html` or `editable_tco_preview.html` in a browser.

On GitHub Pages, publish from the repository root and share the Pages URL. The static page supports calculator interactions in the browser and links to the source-tab previews.

## Local Excel Save-Back

The local `serve_editable_tco_preview.mjs` server can save edits back to the Excel workbook on this machine. That workbook is not included in this repository, and save-back is disabled automatically when the page is hosted as a static GitHub Pages site.

## Source Tabs

The price input and SKU guideline source tabs are included under `sheet_previews/`. Public price references are linked where available; quote-required and assumption-based on-prem values remain editable in the calculator.
